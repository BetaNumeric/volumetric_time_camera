# **volumetric time camera**

Volumetric Time Camera is a browser-based long-exposure camera interface for the Volumetric Time installation. 
It uses the device camera to create layered captures over time, with adjustable blending modes including brightest, mean, and median, plus controls for strength, median frame count, and rolling stack duration. 
Captures can be reviewed in an in-app image folder and downloaded individually or in bulk. 
The app is designed as a lightweight, offline-friendly PWA for installation.

You can view the live camera app here: https://betanumeric.github.io/volumetric_time_camera/
